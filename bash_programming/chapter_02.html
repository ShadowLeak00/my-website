üìú Script Examples

This script checks if a file does not exist using ! -f. If the file is missing, it creates it with touch.

#!/bin/bash
FILENAME="flow_control_with_if.txt"
if [[ ! -f "${FILENAME}" ]]; then
    touch "${FILENAME}"
fi

This example compares two strings using ==. If they are equal, a message confirms the match.

#!/bin/bash
VAR1="nostarch"
VAR2="nostarch"
if [[ "${VAR1}" == "${VAR2}" ]]; then
    echo "They are equal!"
else
    echo "They are not equal!"
fi

This snippet checks if one number is greater than the other using -gt.

#!/bin/bash
VAR1="10"
VAR2="20"
if [[ "${VAR1}" -gt "${VAR2}" ]]; then
    echo "${VAR1} is greater"
else
    echo "${VAR1} is smaller"
fi

üîÅ Loops and Control

This loop prints numbers from 1 to 10. The seq 1 10 generates a sequence, and the for loop iterates through each number, printing it.

for index in $(seq 1 10); do
    echo "${index}"
done

This creates an infinite while loop using true, but the break statement immediately exits the loop after the first iteration. 

while true; do
    echo "Looping..."
    break
done

This loop processes all files starting with example_file. If it matches example_file1, it skips that file using continue. Otherwise, it writes a random number into each file.

for file in example_file*; do
    if [[ "${file}" == "example_file1" ]]; then
        echo "Skipping the first file"
        continue
    fi
echo "${RANDOM}" > "${file}"
done

üîé Filtering with grep, awk

  • The grep command searches for lines containing a specific IP in log.txt.
  • The awk command prints the first and last fields of each line. $1 is the first word; $NF is the last field.
grep "35.237.4.214" log.txt
awk '{print $1, $NF}' log.txt

  • The second prints only lines 2 to 15 from the file.

sed -n '2,15p' log.txt

üìÇ Environment & Aliases

  • The alias command creates a shortcut for a verbose Nmap scan.
  • source ~/.bashrc reloads the Bash config so the alias becomes available immediately.

source ~/.bashrc
